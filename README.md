# Development-of-a-Fake-News-Detection-Web-Application-Using-Machine-Learning
The rapid spread of fake news across digital platforms has increased the need for effective automatic detection systems. This study proposes a hybrid fake news detection approach combining TF-IDF feature extraction with DistilBERT contextual embeddings to improve classification performance. Support Vector Machine, Random Forest, and XGBoost classifiers were applied to labelled datasets of fake and real news collected from open-source repositories. Experimental results showed that the DistilBERT with Support Vector Machine classifier achieved the highest accuracy of approximately 98%. A user-friendly interface was also developed to enable real-time prediction of news authenticity, demonstrating the effectiveness and practical applicability of combining contextual embeddings with traditional machine learning classifiers.
